📖 Ollama Setup Guide (click to expand)

Step 1 — Install Ollama

On Linux / Android (Termux):

curl -fsSL https://ollama.com/install.sh | sh

On macOS / Windows: download from ollama.com

Step 2 — Start Ollama server

ollama serve

Keep this running in a separate terminal. Default port: 11434.

Step 3 — Pull a model

ollama pull llama3.2 # 2GB, fast ollama pull qwen2.5:3b # 2GB, multilingual ollama pull gemma3:4b # 3GB, Google

Verify with: ollama list

Step 4 — Configure MOBOT

  1. Enter the Ollama URL above (default: http://localhost:11434)
  2. Click 🔌 Test Connection — should show ✅
  3. Click 🔄 Fetch Models — select your model
  4. Click ✔ Use this model — fields auto-fill
  5. Click 💾 Save Config at the top
  6. Go to 🤖 Live Agent and start chatting!

📡 Using Ollama on another device (e.g. PC → Android)?

Start Ollama with LAN access on the PC:

OLLAMA_HOST=0.0.0.0 ollama serve

Then set URL to: http://<PC-IP>:11434

✔ Model & API Base auto-filled. Click 💾 Save Config at the top, then go to 🤖 Live Agent.
